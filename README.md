# PixelForge

AI-powered image editing web application.

## Overview
PixelForge is a modern AI image editing platform designed for creators and developers. It provides a modular, scalable architecture for building creative SaaS tools.

## Features
- AI image editing tools
- Background removal
- Image enhancement
- Smart cropping and resizing
- Secure authentication (JWT)
- Responsive UI
- Cloud-ready architecture

## Tech Stack
- Frontend: Next.js / React / Tailwind CSS
- Backend: Node.js / Express
- Database: MongoDB / PostgreSQL
- Auth: JWT / OAuth
- Storage: Cloudinary / AWS S3
- Deploy: Vercel / Render

## Getting Started

### Install
git clone https://github.com/vihanga-nimsara/PixelForge.git
cd PixelForge
npm install

### Run
npm run dev

Open: http://localhost:3000

## Environment Variables
NEXT_PUBLIC_APP_URL=
DATABASE_URL=
JWT_SECRET=
